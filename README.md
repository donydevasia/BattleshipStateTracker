# BattleshipStateTracker
This project contains the implementation of state tracker for a single player in Battleship game.
The assumptions are as given below
1. Board is 10x10
2. Both Row and Column numbers are named 1 to 10
3. There are 5 types of ships that can be added to the board. Carrier with 5 squares, Battleship(4), Cruiser(3), Submarine(3) and Destroyer(2)
4. It takes an attack and reports back Hit or Miss. When all ships are sunk, it reports back so.

Development Environment:-
The solution is developed as Class Library targeting .Net Framework 4.6.1 using Visual Studio 2017. It also contains MSTest project for unit tests.
